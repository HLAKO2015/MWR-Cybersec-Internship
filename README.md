# MWR-Cybersec-Internship (Application Security)
I have enrolled in the Web Application Security Internship by MWR Cybersec. 
This internship is about Web Application Security basics.

From Feb 2026 to May 2026.
The internship is 16 weeks long,
and it provides us with industry practical work.

5. [Cross Site Scripting Part 1 (XSS)](Cross Site Scripting/XSS.md)
6. [XSS and Document Object Model (DOM)](Document Object Model/DOM.md)
7. [Cross Site Request Fogery (CSRF)](Cross Site Request Fogery/CSRF.md)
